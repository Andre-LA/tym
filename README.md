# tym

`tym` is a tiny terminal using .

## Dependencies

- [GTK+3](https://www.gtk.org/)
- [VTE](https://github.com/GNOME/vte)

## Build

```
$ ./configure
$ make
$ sudo make install   // if you want to install
```
